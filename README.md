# GSOUL-地靈傳說

> 南投縣互動式觀光體驗遊戲平台

---

## 專案概述

**GSOUL-地靈傳說** 是一款專為南投縣設計的「遊戲化導覽旅遊平台」，透過 GPS 定位觸發、任務闖關、角色收集與在地商家合作機制，將散落各地的觀光資源整合為「可探索、可互動、可重複體驗」的文化旅程。

本專案以「**永續觀光 × 地方創生 × 數位轉型**」為核心價值，期望成為串聯南投觀光、文化與商業的數位載體。

## 🏛️ 專案執行架構與分工

為確保專案能從「創意提案」轉化為「穩定的公共服務」，建議採行「**政府主導、專業委外、學生協作**」的三方合作模式。

### 角色職責劃分

| 角色 | 定位 | 核心職責 | 為什麼需要？ |
|------|------|---------|-------------|
| **南投縣政府** | **計畫擁有者** | 1. **預算編列與發包**：爭取中央補助或地方預算。<br>2. **跨局處協調**：統合觀光處、計畫處（點點投）資源。<br>3. **驗收與監管**：確保系統目標符合縣政需求。 | 公權力介入才能打通「點點投」系統與商家資源。 |
| **專業資訊廠商** | **主要執行方** | 1. **系統開發與維運**：負責高強度、高資安要求的系統建置與 API 串接。<br>2. **全天候營運**：提供 24/7 伺服器監控與客服支援。<br>3. **資安合規**：確保符合政府資通安全責任等級要求。 | 學生團隊難以負擔大規模系統的維運、資安風險與賠償責任。 |

## 🚀 專案啟動流程

建議採用**三階段漸進式**導入，降低風險並確保成效：

#### 第一階段：概念驗證期 (POC) - **現在**
*   **執行者**：**專業廠商 (小額採購)** 搭配 **學生團隊 (內容授權)**
*   **目標**：驗證核心玩法與系統穩定性。
*   **範圍**：由政府進行 10 萬元以下小額採購，委託小型工作室或個人開發者，將學生團隊的 Demo 概念優化為可穩定運作的 Beta 版。學生團隊僅需提供圖片素材與設計理念，**不需負擔程式開發工作**。
*   **政府動作**：提供行政指導，協助對接 3-5 家種子商家，並進行小規模試營運。

#### 第二階段：示範導入期 (Pilot) - **預計 3-6 個月**
*   **執行者**：**專業廠商 (政府發包)** + **學生團隊 (顧問)**
*   **目標**：完成與「南投點點投」系統對接，測試點數經濟循環。
*   **範圍**：擴大至 4 條主題路線，全面介接點點投特約商店資料庫。
*   **政府動作**：正式編列預算或專案招標，舉辦「GSOUL 上線記者會」，結合大型節慶（如花火節）推廣。

#### 第三階段：全面擴散期 (Scale-up) - **預計 1 年後**
*   **執行者**：**專業廠商 (長期維運)**
*   **目標**：深化數據應用，建立自我造血的商業模式。
*   **範圍**：開放全縣商家自主申請上架，甚至授權其他縣市加盟。
*   **政府動作**：轉型為平台監管方，利用數據優化縣政決策。

## 💰 預算與招商規劃

### 預算估算 (Pilot 階段參考)

本計畫具備**低建置成本、高擴充效益**特性。初期建議以**專案委託**或**創新補助**方式進行。

| 項目 | 預算比例 | 內容說明 |
|------|---------|---------|
| **系統開發與整合** | 40% | 遊戲功能優化、後端資料庫建置、**「點點投」API 介接工程** (關鍵成本)。 |
| **內容與美術製作** | 25% | 40+ 景點故事撰寫、角色/道具繪製、AR 場景建模。 |
| **行銷與活動推廣** | 20% | 上線活動辦理、社群廣告投放、KOL 合作、實體文宣。 |
| **營運與商家輔導** | 15% | 商家說明會、客服系統運作、伺服器維運費用。 |

*預估總經費：約新台幣 150萬 - 300萬 (視功能複雜度與活動規模而定)*

### 招商與商家管理策略

**Q: 需要重新大規模招商嗎？**
**A: 初期不需要，直接槓桿現有資源。**

1.  **Level 1：既有特約商店 (無痛導入)**
    *   直接匯入「南投點點投」現有的特約商店名單。
    *   **優勢**：商家已有核銷設備與經驗，系統介接後即可上線，無需重新簽約。

2.  **Level 2：重點觀光熱點 (主動開發)**
    *   針對遊戲路線上的指標性景點（如日月潭纜車、九族文化村、清境農場）進行專案洽談。
    *   **誘因**：提供客製化的「專屬任務關卡」（如：尋找纜車怪獸），加深品牌置入。

3.  **Level 3：在地小微店家 (開放申請)**
    *   中後期開放一般店家線上申請成為「任務點」。
    *   **門檻**：需加入成為「點點投」特約商，間接推動縣府點數平台的普及率。

## 政府合作與應用策略

本專案不僅是一個觀光導覽工具，更可作為南投縣政府推動數位治理與地方經濟的重要介面。

### 1. 政策行銷與公民參與的新入口
- **遊戲化政策宣導**：將生硬的政策資訊轉化為遊戲中的「知識問答」或「尋寶任務」。例如，在新建公共建設旁設置任務點，民眾抵達打卡並回答相關問題即可獲得獎勵，有效提升政策觸及率與民眾參與感。
- **數位縣民證整合**：未來可與南投數位縣民平台介接，GSOUL帳號即為數位分身，強化對政府數位服務的黏著度。

### 2. 「南投點點投」深度整合計畫
本專案將與南投縣政府現有的**南投點點投**數位點數平台進行深度戰略結合，創造「1+1>2」的綜效：

| 整合面向 | 執行與效益 |
|---------|------------|
| **點數生態系串聯** | **GSOUL作為「集點端」**：民眾透過GSOUL完成深度旅遊任務（如走訪偏鄉部落、參與環保淨山），獲取遊戲積分。 <br> **點點投作為「核銷端」**：遊戲積分可按比例兌換為「南投點點投」點數，直接在全縣特約商店折抵消費（1點折1元），實現「玩遊戲賺現金」的實質回饋。 |
| **共享特約商家網** | 透過介接「點點投」既有的特約商店資料庫（涵蓋觀光工廠、商圈美食、旅宿等），GSOUL可快速擴充任務點與獎勵兌換點，商家亦無需重複導入新系統，即可獲得遊戲帶來的精準觀光人流。 |
| **數位共好生態圈** | 落實「全額支付、零手續費」精神，遊戲引導的消費直接回饋在地店家，政府則透過平台掌握觀光經濟數據。 |

### 3. 數據驅動的觀光治理
- **人流熱力圖分析**：透過玩家GPS足跡，分析熱門/冷門景點分佈與滯留時間，協助觀光處優化交通接駁與設施配置。
- **精準行銷決策**：分析不同路線（如「手作文化」vs「農食永續」）的玩家輪廓，作為未來規劃特色旅遊行程的數據基礎。

## 預期效益

### 對應 SDGs 目標
*   **8.9 永續觀光**：促進在地文化與產品的永續旅遊。
*   **9.B 產業創新**：支持國內技術開發，推動觀光數位轉型。
*   **11.4 文化保存**：透過遊戲故事化，強化對文化與自然遺產的保護。

### 行銷推廣與呈現策略

為觸及不同年齡層旅客，將採「**全通路**」策略，結合線上社群擴散與線下實體引導。

#### 1. 呈現載體：降低門檻，即掃即玩
*   **網頁應用程式 (PWA)**：**無需下載 App**，遊客只需掃描景點告示牌上的 QR Code 即可直接進入遊戲，大幅降低使用門檻。
*   **LINE 官方帳號整合**：利用 LINE Chatbot 作為任務提示助手與點數查詢入口，將遊戲嵌入台灣人最常用的通訊軟體中，無需額外學習操作。
*   **IG 濾鏡**：開發 3-5 款結合南投特色（如：茶葉精靈、原民圖騰）的 AR 濾鏡，遊客拍照後分享至限時動態即可完成「隱藏任務」，利用 UGC (使用者創作內容) 達到病毒式傳播。

#### 2. 推廣管道：分眾行銷
| 管道 | 目標族群 | 執行方式 |
|------|---------|---------|
| **南投點點投 LINE / APP** | **既有會員** | 透過推播功能，精準通知既有點數會員參與新任務，主打「玩遊戲賺更多點數」。 |
| **KOL / 網紅行銷** | **年輕遊客 (18-35歲)** | 邀請旅遊 YouTuber 或 IG 網紅（如：林林夫妻、好想去南投）拍攝「一日南投尋寶挑戰」Vlog，示範遊戲玩法與獎勵兌換過程。 |
| **實體場域策展** | **親子家庭 / 團客** | 於**南投休息站**、**遊客中心**設置大型互動裝置或電子看板，以「掃碼領 50 元見面禮」為誘因，引導遊客在進入南投的第一站就啟動遊戲。 |
| **節慶限定活動** | **全體遊客** | 配合**南投燈會**、**日月潭花火節**，推出「期間限定關卡」，將實體燈區變為遊戲任務點，增加逛展樂趣。 |
